# Proyecto ATD

Repositorio organizado para el proyecto académico ATD.

## ?? Estructura del Repositorio

### **pagina-web/**
Contiene todos los archivos para la **página web funcional** que se despliega en Render:
- Archivos PHP
- HTML, CSS, JavaScript
- Imágenes y recursos multimedia
- Configuración para despliegue

### **request/**
Contiene el **código fuente y análisis** del proyecto:
- Jupyter Notebooks (.ipynb)
- Scripts de procesamiento (Python, R, etc.)
- Conjuntos de datos
- Documentación técnica

## ?? Despliegue de la Página Web
1. **Conectar Render** a este repositorio
2. **Root Directory**: /pagina-web
3. **Runtime**: PHP
4. **Start Command**: php -S 0.0.0.0:$PORT

## ?? Análisis de Datos
- Los notebooks y scripts están en /request
- Usar Jupyter o entornos de desarrollo para ejecutar
